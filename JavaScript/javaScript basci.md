## 对象
* 访问对象属性的语法：`ObjectName.propertyName`
* 访问对象方法的语法：`ObjectName.methodName()`

### Date 日期对象

语法：`var date = new Date();`

`date`成为了日期对象，并且有了当前时间作为初始值；

如果自定义初始值有如下方法：

```js
var date = new Date(2012, 10, 1)
```
